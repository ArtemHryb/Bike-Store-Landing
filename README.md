# BIKE Landing Page

This project is a responsive landing page created based on the BIKE New Version design mockup from Figma. Its purpose is to present a modern and visually engaging promotional page for a bike product line, highlighting key features, visual content, and calls to action. The landing page focuses on clean UI, smooth interactions, and an adaptive layout to ensure a consistent user experience across devices.
